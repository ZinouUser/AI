Note setup audio Pi5
Le Pi5 dispose de 4 ports USB (2× USB 3.0 + 2× USB 2.0). Configuration recommandée pour la démo : USB 2.0 port 1 → Carte son USB (sortie TTS Piper)
USB 2.0 port 2 → Micro RØDE NT-USB Mini (entrée STT)
USB 3.0 port 1 → Clé USB backup / hub
USB 3.0 port 2 → Libre Dans le code Python : spécifier le device micro avec AZURE_STT_MIC_DEVICE_ID récupéré via arecord -l
